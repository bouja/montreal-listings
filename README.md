# Montreal Listings

Projet Next.js pour Montréal Listings - plateforme de locations et ventes immobilières.

## Installation locale

1. Cloner le dépôt :

git clone https://github.com/ton-compte-github/montreal-listings.git
cd montreal-listings

2. Installer les dépendances :

npm install

3. Lancer le serveur de développement :

npm run dev

4. Ouvrir [http://localhost:3000](http://localhost:3000) dans le navigateur.

---

## Déploiement sur Vercel

1. Créer un compte sur [vercel.com](https://vercel.com) si ce n’est pas déjà fait.

2. Connecter votre compte GitHub à Vercel.

3. Importer le repository `montreal-listings` dans Vercel.

4. Vercel détecte automatiquement le projet Next.js, configurer (optionnel) et déployer.

5. Le site sera accessible sur l’URL fournie par Vercel.

---

## Personnalisation

- Remplacer l’ID Google Analytics dans `utils/analytics.js` (`G-XXXXXXX`) par votre propre ID.

- Modifier les liens affiliés dans `pages/index.js`.

---

## Commandes utiles

- `npm run dev` : serveur local

- `npm run build` : build pour production

- `npm run start` : lancer la version buildée en local
